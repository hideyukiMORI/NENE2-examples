Priority queue, retry logic in repository layer, retry_count/max_retries, idempotency_key UNIQUE.

## Run

```bash
composer install
composer test        # PHPUnit  (--testdox)
composer analyse     # PHPStan level 8
composer cs          # PHP CS Fixer dry-run
```
