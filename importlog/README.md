str_getcsv $escape fix, partial success, within-batch duplicate detection, CRLF handling.

## Run

```bash
composer install
composer test        # PHPUnit  (--testdox)
composer analyse     # PHPStan level 8
composer cs          # PHP CS Fixer dry-run
```
